# Quiz Game App

Dies ist eine Quiz Game App, die als Übungsprojekt für React erstellt wurde. Die App bietet verschiedene Quiz-Kategorien (z. B. Capitals, Science, Movies) und mehrere Schwierigkeitsstufen, um grundlegende Konzepte in React zu üben.

## Features

- **Mehrere Kategorien:** Wähle zwischen verschiedenen Themenbereichen.
- **Verschiedene Levels:** Jede Kategorie ist in mehrere Schwierigkeitsgrade unterteilt.
- **Interaktive Benutzeroberfläche:** Dynamische Anzeige von Fragen, Auswahlmöglichkeiten und sofortiges Feedback.
- **Modernes Styling:** Dunkles Design mit Goldakzenten und responsivem Layout.
